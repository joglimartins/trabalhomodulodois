Grade Control API
hppt://localhost:3000/
/grades

POST

Endpoint para criar novo registro no arquivo grades.json

Exemplo de envio: { "student": "Maria José", "subject": "NodeJs", "type": "Desafio", "value": 10 }
/grades

PUT

Endpoint para atualizar uma grade no arquivo grades.json

Exemplo de envio: { "id": 49, "student": "Jogli Martins", "value": 25 }
/grades

DELETE

Endpoint para excluir uma grade no arquivo grades.json

Exemplo de envio: { "id": 49 }
/grades/get/{id}

GET

Endpoint para consultar uma grade no arquivo grades.json

Exemplo de envio: {}
/grades/total

GET

Endpoint e Função para consultar o total das notas de um student em uma subject no arquivo grades.json

Exemplo de envio: { "subject": "01 - JavaScript", "type": "Fórum" }
/grades/avg

GET

Endpoint e Função para consultar a media das notas das grades de um determinado subject e type no arquivo grades.json

Exemplo de envio: { "subject": "01 - JavaScript", "type": "Fórum" }
